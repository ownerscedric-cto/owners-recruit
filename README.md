# 오너스경영연구소 스마트 리크루팅 시스템

효율적이고 안전한 입사 프로세스를 통해 인재와 기업을 연결하는 디지털 게이트웨이

## 🚀 기능

### 지원자용 페이지
- **입사 가이드**: 프로세스 안내, 필요 서류, 교육 정보
- **정보 입력**: 단계별 입력 폼, Progress Bar, 파일 업로드
- **진행 상황 확인**: 실시간 상태 추적, 메시지 확인

### 인사팀용 페이지
- **대시보드**: 실시간 통계, 차트, 최근 지원자
- **지원자 관리**: 목록 조회, 필터링, 승인/반려 처리
- **보고서**: 월별 성과, 모집인별 실적, 각종 통계

### 관리자용 페이지
- **시스템 설정**: 일반 설정, 폼 필드 관리, 서류 유형 설정
- **사용자 관리**: 권한 관리, 사용자 추가/삭제
- **알림 및 백업 설정**: 이메일/SMS 알림, 자동 백업

## 🛠 기술 스택

- **Framework**: Next.js 15 (App Router, Server Components)
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI
- **Icons**: Lucide React
- **Charts**: Recharts
- **Language**: TypeScript

## 📱 주요 특징

### 사용자 경험 (UX)
- **모바일 우선**: 반응형 디자인으로 모든 디바이스 지원
- **직관적 네비게이션**: 명확한 메뉴 구조와 진행 상황 표시
- **접근성**: WCAG 준수, 키보드 네비게이션 지원

### 개발자 경험 (DX)
- **Server Components**: 성능 최적화와 SEO 향상
- **TypeScript**: 타입 안전성과 개발 효율성
- **컴포넌트 기반**: 재사용 가능한 UI 컴포넌트
- **모던 개발 환경**: ESLint, Prettier, Hot Reload

## 🏗 프로젝트 구조

```
src/
├── app/                    # App Router 페이지
│   ├── applicant/         # 지원자용 페이지
│   │   ├── apply/         # 정보 입력
│   │   ├── guide/         # 입사 가이드
│   │   └── status/        # 진행 상황
│   ├── admin/             # 인사팀용 페이지
│   │   ├── dashboard/     # 대시보드
│   │   ├── applicants/    # 지원자 관리
│   │   └── reports/       # 보고서
│   └── manager/           # 관리자용 페이지
│       └── settings/      # 시스템 설정
├── components/            # 재사용 컴포넌트
│   ├── ui/               # 기본 UI 컴포넌트
│   ├── forms/            # 폼 컴포넌트
│   ├── dashboard/        # 대시보드 컴포넌트
│   └── shared/           # 공통 컴포넌트
├── lib/                  # 유틸리티 함수
└── types/                # TypeScript 타입 정의
```

## 🚀 실행 방법

### 개발 환경 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

http://localhost:3000 에서 애플리케이션을 확인할 수 있습니다.

### 빌드 및 배포

```bash
# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

## 📄 페이지 가이드

### 메인 페이지 (/)
- 시스템 개요 및 주요 기능 소개
- 역할별 입구 제공 (지원자/인사팀/관리자)

### 지원자 영역 (/applicant/)
- `/applicant/guide`: 입사 프로세스 및 필요 서류 안내
- `/applicant/apply`: 단계별 정보 입력 및 서류 업로드
- `/applicant/status`: 진행 상황 및 메시지 확인

### 인사팀 영역 (/admin/)
- `/admin/dashboard`: 실시간 대시보드 및 통계
- `/admin/applicants`: 지원자 목록 및 승인 관리
- `/admin/reports`: 각종 보고서 및 분석 자료

### 관리자 영역 (/manager/)
- `/manager/settings`: 시스템 전반 설정 관리

## 🎨 디자인 시스템

### 색상 팔레트
- **Primary**: 파란색 계열 (신뢰, 전문성)
- **Success**: 녹색 계열 (승인, 완료)
- **Warning**: 황색 계열 (대기, 주의)
- **Error**: 빨간색 계열 (반려, 오류)

### 타이포그래피
- **헤딩**: Inter 폰트, 다양한 크기
- **본문**: 시스템 폰트, 가독성 최적화
- **버튼**: 중간 굵기, 명확한 CTA
